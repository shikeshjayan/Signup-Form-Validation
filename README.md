# Signup Form Validation

A fully responsive and interactive Signup Form Validation is Front-End project built with HTML, CSS, and JavaScript.
It includes real-time form validation, password strength rules, password visibility toggle, a success popup, and accessibility-friendly features.

![Signup Form Screenshot](assets/signup-form-validation.png)

## Installation

No installation required. Simply clone or download the project, and open the `index.html` file in any modern web browser.

`git clone https://github.com/shikeshjayan/Signup-Form-Validation.git`
`cd Signup-Form-Validation`
`open index.html`

## Usage

Open `index.html` in a browser and fill in the form fields:

- Fullname: Must be 3-25 characters without special characters.
- Email: Must be in valid email format.
- Password: Minimum 8 characters, including uppercase, lowercase, number, and special character.
- Confirm Password: Must match the password.
- Terms & Conditions: Must be accepted before submitting.

The form provides real-time validation feedback and only allows submission when all conditions are met. After submission, a success popup with a personalized welcome message is shown.

## Features

- Responsive design using Bootstrap 5 and custom CSS.
- Real-time validation with detailed error messages.
- Show/hide password toggle.
- Terms & Conditions enforcement.
- Success popup upon valid registration.
